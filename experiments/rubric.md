# Scoring Rubric

Score each generated image on these 5 dimensions. Use **integers 1–5** (no half-points). Total = sum / 25.

Reviewers should see all variants for the same page side-by-side before scoring, to anchor judgment.

## 1. Character consistency
*How recognizable are the protagonist (and recurring characters) compared to other pages of the same run?*
- **5** — Identical across pages: same species, build, colors, clothing, facial features.
- **4** — Same identity, minor drift in 1 attribute (e.g., shirt color shifts).
- **3** — Recognizable but with noticeable inconsistencies (proportions, palette).
- **2** — Hard to tell it's the same character without context.
- **1** — Looks like a different character.

## 2. Scene clarity
*Does the image visually communicate the moment described in the page text?*
- **5** — Subject + action + setting unmistakably match the page.
- **4** — Match is clear, with a small omission or ambiguity.
- **3** — Roughly right but key scene element is missing or wrong.
- **2** — Tangentially related to the page.
- **1** — Doesn't depict the page at all.

## 3. Emotional expression
*Is the emotional tone of the page conveyed (through posture, light, environment — NOT face only)?*
- **5** — Tone reads instantly and supports the therapeutic intent.
- **4** — Tone reads, slightly understated/overstated.
- **3** — Tone is neutral when it should be specific.
- **2** — Tone contradicts the page text in subtle ways.
- **1** — Wrong tone entirely (e.g., cheerful when scared).

## 4. Age-appropriateness
*Suitable for the target age range (3–5, 6–8, etc.)? No frightening, mature, or confusing imagery.*
- **5** — Perfectly tuned to the age range.
- **4** — Suitable, slightly off (a touch too abstract / too busy).
- **3** — Acceptable but parents might pause.
- **2** — Inappropriate for the age (too scary, too text-heavy, etc.).
- **1** — Unusable for the target audience.

## 5. Art quality
*Coherent illustration: anatomy, composition, lighting, no rendering artifacts.*
- **5** — Polished children's-book quality.
- **4** — Strong, with one small flaw (extra finger, soft edge).
- **3** — Acceptable but visibly AI-generated.
- **2** — Distracting artifacts (warped faces, melted hands).
- **1** — Broken / unusable output.

## Reporting

For each experiment, fill in [`report-template.md`](report-template.md). Total score per page is the sum of the 5 dimensions (out of 25). Average across pages for the experiment's overall score.
